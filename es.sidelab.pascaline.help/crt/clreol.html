ClrEol

Clear from cursor position till end of line.

Declaration

Source position: crth.inc line 79

procedure ClrEol;

Description

ClrEol clears the current line, starting from the cursor position, to the end of the window. The cursor doesn't move

Errors

None.

See also

DelLine

  

Delete line at cursor position.

InsLine

  

Insert an empty line at cursor position

ClrScr

  

Clear current window.

Example

Program Example9;
uses Crt;

{ Program to demonstrate the ClrEol function. }
var
  I,J : integer;

begin
  For I:=1 to 15 do
    For J:=1 to 80 do
      begin
      gotoxy(j,i);
      Write(j mod 10);
      end;
  Window(5,5,75,12);
  Write('This line will be cleared from',
        ' here till the right of the window');
  GotoXY(27,WhereY);
  ReadKey;
  ClrEol;
  WriteLn;
end.
